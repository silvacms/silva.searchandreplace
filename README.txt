======================
silva.searchandreplace
======================

``silva.searchandreplace`` let you add a new `Silva`_ service *Silva
Search and Replace Service* for Silva 2.1 and higher that is able to
search inside multiple documents, and replace found strings with a new
provided value.

This extension doesn't work in Silva 3 with the documents created with
``silva.app.document``.

Code repository
===============

The code for this extension can be found in Git:
https://github.com/silvacms/silva.searchandreplace

.. _Silva: http://silvacms.org
